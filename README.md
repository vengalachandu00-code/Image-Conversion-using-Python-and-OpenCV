# Image Processing and Conversion using Python & OpenCV

## 📌 Project Overview
This project is developed using Python and OpenCV to perform various image processing operations. It allows users to load an image and apply different conversions and transformations.

## 🚀 Features
- Load and display images
- Convert image to Grayscale
- Convert image to Binary (Black & White)
- Resize images
- Image color transformations
- Simple and user-friendly implementation
- Object-Oriented Programming (OOP) approach

## 🛠️ Technologies Used
- Python
- OpenCV
- NumPy
- Jupyter Notebook
